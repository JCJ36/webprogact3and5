# Activity 3 and 5 in one Project.

This project is a sample Forum Site named Re:Edit with post and user moderation.

# Setup

To setup the project on XAMPP, place the forum folder in C:\xampp\htdocs and on phpMyAdmin, run setup.sql to setup the database.

# Information

While accounts can be registered on the page, setting admins and moderators have to be toggled on phpMyAdmin for any existing users the database. 

While creating users directly in phpMyAdmin is possible, preferably creating an account on the register page and then toggling them to be the desired role is more stable.
